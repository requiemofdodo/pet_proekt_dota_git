# DOTA 2 Hero Manager

Приложение для управления героями DOTA 2.

## Возможности
- Добавлять новых героев
- Добавлять навыки к героям
- Показывать всех героев
- Удалять героев

## Как запустить
python main.py

## Как запустить тесты
python test_main.py

## Структура проекта
- `hero.py` - функции для работы с героями
- `manager.py` - функции для управления героями
- `main.py` - интерактивное меню
- `test_main.py` - тесты
